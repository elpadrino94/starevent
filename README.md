STAREVENT est une plateforme web développée avec Django qui facilite la découverte et la publication d'événements locaux.

Fonctionnalités principales :

Pour les utilisateurs : 
Parcourir et rechercher des événements publiés par catégories (Musique, Sport, Culture, Tech), avec détails complets incluant images, descriptions et localisations.
Pour les promoteurs et organisateurs : Créer, publier et gérer leurs événements via un système d'authentification sécurisé. Les événements peuvent inclure des images, dates, lieux et descriptions détaillées.

Interface utilisateur : 
Design responsive avec Bootstrap, navigation intuitive et gestion des comptes utilisateurs.

Technologies utilisées :
Backend : Django (Python)
Base de données : SQLite
Frontend : HTML, CSS, JavaScript, Bootstrap
Authentification : Système intégré de Django avec rôles personnalisés (Utilisateur / Promoteur)
Gestion des médias : Upload d'images pour événements et profils utilisateurs
Ce projet vise à connecter les passionnés d'événements avec les organisateurs, en offrant une expérience simple et efficace pour découvrir et partager des moments uniques. Idéal pour les communautés locales cherchant à promouvoir leur culture et leurs activités
